已導入能力

  • 環境變數控制 `DEBUG`、`ALLOWED_HOSTS`、Cookie 安全、HSTS 與 proxy header。
  • 快取支援 `locmem` / `filebased`，後續可再切換 Redis。
  • 靜態檔、上傳、快取與 log 目錄會自動建立。
  • 每個 request 會附帶 `X-Request-ID`，並寫入 access log。
  • `/api/v1/` 與登入相關寫入有 cache-based rate limiting。
  • 提供 liveness / readiness health check。
